## 核心知识点

### 1. 参数化（路径不写死）

**操作：** 管理参数 → 新建参数 → 类型：文本 → 当前值：文件夹路径

**改前：**
```
Excel.Workbook(File.Contents("C:\固定路径\文件.xlsx"), null, true)
```

**改后：**
```
Excel.Workbook(File.Contents(文件夹路径 & "\文件名.xlsx"), null, true)
```

**财务类比：** 参数 = PQ里的"单元格引用"，写死路径 = Excel里直接写数字。

---

### 2. 参数表驱动合并

**参数表结构（Excel工作表）：**

| 公司名称 | 文件名 | 纳入合并 |
|----------|--------|----------|
| 海澜餐饮 | 海澜餐饮_利润表.xlsx | Y |
| 绿茶茶饮 | 绿茶茶饮_利润表.xlsx | Y |

**操作链条：**
1. 参数表 Ctrl+T 转表格
2. PQ 从表格导入 → 筛选纳入合并=Y
3. 添加"完整路径"列：`文件夹路径 & [文件名]`
4. 添加"文件内容"列：`Excel.Workbook(File.Contents([完整路径]), null, true)`
5. 两次展开（先展Data，再展实际列）
6. 清洗（去表头、重命名、删冗余列）
7. 透视成宽表

**使用方法：**
- 加公司 → 参数表添一行 + 放文件 → 刷新
- 删公司 → 参数表改Y为N → 刷新

---

### 3. 🔥 动态合计公式（今日高光）

```
List.Sum(List.Transform(List.Skip(Record.FieldValues(_), 1), each _ ?? 0))
```

| 部分 | 作用 | Excel类比 |
|------|------|-----------|
| Record.FieldValues(_) | 取整行所有字段值 | 抓出整行数据 |
| List.Skip(..., 1) | 跳过第1个（项目名） | 跳过A列 |
| each _ ?? 0 | null转0 | 空格当0 |
| List.Sum(...) | 全部求和 | =SUM(B2:ZZ2) |

**为什么重要：** 一个列名都不写！3家5家30家公司，自动取所有数字列求和。

## 遇到的问题及解决

| 问题 | 原因 | 解决 |
|------|------|------|
| 路径拼接少反斜杠 | 参数值结尾无 `\` | 参数值以 `\` 结尾 |
| 透视后合计 null | null + 数字 = null | 替换 null 为 0 + 动态公式 |
| 公司变了合计报错 | 公式写死列名 | 改用动态合计公式 |
| 新公司出现表头行 | 文件首行混入 | 筛选"项目"列排除表头 |
| 参数表改了不生效 | 没做筛选步骤 | PQ里"筛选的行"只留Y |
