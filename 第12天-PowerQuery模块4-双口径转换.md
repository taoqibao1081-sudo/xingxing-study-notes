## 一、核心概念：为什么需要双口径转换

### 现实场景
50家子公司不可能都用同一套会计准则：
- 小规模子公司 → 《小企业会计准则》（会小企01表）→ 科目简化
- 大型子公司/母公司 → 《企业会计准则》（会企01表）→ 科目更全

两套准则的**科目名称、数量、排列都不同**，不先统一口径直接合并就是张冠李戴。

### 财务类比
> 就像3家用的是"小企业科目编码表"，1家用的是"企业科目编码表"。合并前必须把企业准则那张表翻译成小企业科目，才能放一起加总。

## 二、两大准则科目差异对照

### 典型差异一览

| 企业准则（会企01表） | 小企业准则（会小企01表） | 差异类型 |
|---|---|---|
| 交易性金融资产 | 短期投资 | 名称不同，本质相同 |
| 预付款项 | 预付账款 | "项"→"账" |
| 应收股利/应收利息（并入其他应收款） | 应收股利、应收利息（单列） | 企业合并，小企业拆分 |
| 合同资产 | （无） | 企业专属·新收入准则 |
| 使用权资产 | （无） | 企业专属·新租赁准则 |
| 固定资产（净值） | 固定资产原价/累计折旧/账面价值 | 企业不拆分三行 |
| 债权投资 | 长期债券投资 | 名称不同 |
| 其他权益工具投资 | （无） | 企业专属 |
| 其他综合收益 | （无） | 企业专属 |
| 递延所得税资产/负债 | （无） | 企业专属·小企业不确认 |
| 商誉 | （无） | 企业专属 |
| 合同负债 | （无） | 企业专属·新收入准则 |
| 租赁负债 | （无） | 企业专属·新租赁准则 |
| 预收款项 | 预收账款 | "项"→"账" |

### 三种映射类型

| 类型 | 含义 | 示例 |
|---|---|---|
| **同名映射** | 两个准则名称一致，直接对应 | 货币资金→货币资金 |
| **名称转换** | 叫法不同，本质同一回事 | 交易性金融资产→短期投资 |
| **企业独有** | 小企业准则没有这个科目 | 合同资产→归入"其他流动资产" |

## 三、核心交付物：科目映射表（字典）

文件路径：`桌面\WB练习\合并\科目映射表.xlsx`

结构：7列 × 55行映射记录

| 企业准则科目 | 企业行次 | 映射类型 | 小企业准则科目 | 小企业行次 | 合并口径科目 | 备注 |
|---|---|---|---|---|---|---|
| 交易性金融资产 | 2 | 名称转换 | 短期投资 | 2 | 短期投资 | ... |

**设计要点**：
- "合计行"和"分类标题行"（如"流动资产："）不参与映射
- "企业独有"科目归入最接近的小企业科目（如合同资产→其他流动资产）
- 其他应收款在映射表里有3条记录（对应小企业3个拆分科目），后续需去重

## 四、Power Query 实操流程

### 完整操作链

```
企业准则报表（海澜公司）
    ↓ 1. 加载到Power Query
    ↓ 2. 删除前4行表头 → 提升标题
    ↓ 3. 拆分双栏（资产侧 / 负债权益侧）
    ↓ 4. 载入科目映射表为第二个源
    ↓ 5. 合并查询（资产侧科目 匹配 映射表·企业准则科目）
    ↓ 6. 展开小企业准则科目列
    ↓ 7. 自定义列：if null then 原名 else 翻译名
    ↓ 8. 去重（处理一对多映射的重复行）
    ↓ 9. 删除旧列，只保留：小企业科目 | 期末余额 | 年初余额
    ↓ 10. 负债权益侧重复步骤4-9
    ↓ 11. 追加合并（资产侧 + 负债权益侧）
    ↓ 12. 验证平账 → 上载到Excel
```

### 关键M代码逻辑（自定义列公式）

```
if [合并口径科目] = null then [资  产] else [合并口径科目]
```

含义：映射表有翻译结果就用翻译结果，null的行（合计行/分类标题）保留原名。

### 去重关键

"其他应收款"在映射表里对应"应收股利"+"应收利息"+"其他应收款"3条记录，展开后产生3行重复金额。按住Ctrl同时选中科目列和期末余额列 → 删除重复项。
