## 三、关键操作与避坑

### 1. 数据接入
- 主页 → 获取数据 → Excel → 选择文件
- 建议点「转换数据」进 Power Query 确认，再「关闭并应用」
- Power BI 内置 PQ，和 Excel 里学的一模一样

### 2. KPI 卡片
- 卡片字段拖「期末余额」
- 必须加两个筛选器：**期间** + **科目**
- 例：资产总计 = 期间=2025年12月，科目=资产总计

### 3. 资产负债表矩阵
- 行：科目类别 → 科目（分层）
- 值：期末余额
- **一定要关闭矩阵总总计**，否则明细+小计+总计会重复计算
- 用「科目类别顺序」「科目顺序」列控制显示顺序
- 用「是否展示」列排除辅助项（固定资产原价、累计折旧）

### 4. 资产构成饼图
- 图例：科目；值：期末余额
- 筛选器：科目类别=资产，科目层级≠总计

### 5. 切片器联动
- 删除每个图表里的固定「期间」筛选器
- 插入「期间」切片器和「公司」切片器
- 一个切片器驱动所有图表

### 6. DAX 同比（已完成，2026-07-05 新建测试文件验证）
- 在练习数据 V4 中新增独立的「年份」整数字段
- 度量值公式：
  ```dax
  资产总计_2024 = CALCULATE(SUM('资产负债表'[期末余额]), '资产负债表'[年份]=2024, '资产负债表'[科目]="资产总计")
  ```
- 同比增长率、同比增长额度量值均验证通过

### 7. 条件格式红涨绿跌（2026-07-12 完成）
- 选中同比卡片 → 设置视觉对象格式（油漆刷） → 标注 → 值 → 颜色 → 点 `fx`
- 格式样式选「规则」，基于字段选当前度量值（如 `资产总计_同比增长率`）
- 规则 1：如果值 `> 0` 且 `< 最大`，颜色选红色 `#FF0000`（涨）
- 规则 2：如果值 `> 最小` 且 `< 0`，颜色选绿色 `#00AA00`（跌）
- 说明：Power BI 规则默认是"上下界"格式，所以用 `> 0` 到 `最大` 表示正数区间，用 `> 最小` 到 `< 0` 表示负数区间

## 四、今天踩过的坑

| 问题 | 原因 | 解决 |
|------|------|------|
| KPI 卡片显示 7 百万 | 没固定期间，两期求和了 | 加「期间」筛选器 |
| 矩阵底部总计 2140 万 | 矩阵自动总总计重复计算 | 关闭矩阵「行小计/总计」 |
| 饼图里混进负债/权益 | 没加「科目类别=资产」筛选 | 加筛选器 |
| 同比卡片报错/为0 | 日期字段在 DAX 中匹配异常 | 增加独立「年份」整数字段（已验证，新建测试文件解决） |
| 直接脚本注入 .pbix 条件格式 | Power BI 内部 JSON schema 不公开，导致文件损坏 | 改为手动在格式窗格中设置规则 |
