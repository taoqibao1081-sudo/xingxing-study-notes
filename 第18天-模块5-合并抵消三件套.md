## 场景1：内部往来抵消

**逻辑**：海澜应收绿野50万 ↔ 绿野应付海澜50万。同一笔钱AB面，不能重复计算。

**PQ公式**：
```
if [科目] = "应收账款-绿野茶饮" then -[金额]
else if [科目] = "应付账款-海澜餐饮" then -[金额]
else [金额]
```

**验证**：SUMIFS(E:E,"内部往来抵消*") = **-1,000,000** ✓（双方对冲清零）

---

## 场景2：内部交易抵消

**逻辑**：海澜对绿野销售收入30万 ↔ 绿野向海澜采购成本30万。集团内部转移，非对外交易。

**PQ公式**：
```
if Text.Contains([项目], "内部") then -[金额] else [金额]
```

**验证**：SUMIF(E:E,"内部交易抵消") = **-600,000** ✓（收入-30万+成本-30万）

> 严谨性检验：海澜计收入30万（绿野为交易对手），绿野计成本30万（海澜为交易对手）。合并视角下此30万属集团内部资源转移，不构成对外交易。收入与成本各减30万，利润总额不受影响（收入减30万同时成本减30万），逻辑自洽。

---

## 场景3：股权投资抵消

**逻辑**：海澜持有绿野80%股权，长投800,000。绿野所有者权益合计1,400,000（实收资本100万+未分配利润40万）。海澜占80%=1,120,000，少数股东占20%=280,000。

**PQ公式**：
```
if [科目] = "长期股权投资-绿野茶饮" then -800000
else if [科目] = "实收资本" and [公司名] = "绿野茶饮" then 200000
else if [科目] = "未分配利润" and [公司名] = "绿野茶饮" then 80000
else [金额]
```

**抵消后**：
- 长投 → 0（全额冲掉）
- 实收资本（绿野）→ 200,000（20%少数）
- 未分配利润（绿野）→ 80,000（20%少数）
- 少数股东权益合计 → 280,000

> 严谨性检验：海澜长投800,000 = 绿野实收资本1,000,000×80%，金额配比准确。少数股东权益 = 实收资本200,000 + 未分配利润80,000 = 280,000 = 1,400,000×20%，比例和金额均正确。合并报表中"长期股权投资"归零、"实收资本"仅保留少数份额，符合企业合并准则抵消要求。

## 核心要诀

- **本质**：内部科目 = 左手倒右手，合并视角下不存在
- **方法**：负数对冲法（内部科目金额变负）
- **验证**：SUMIF对冲科目合计 → 应为负数/特定值
- **追踪**：永远保留"抵消说明"列，便于审计
