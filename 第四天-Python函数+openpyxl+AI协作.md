## 一、lesson07：函数 — 把重复代码打包成"小工具"

### 核心语法

```python
def 函数名(参数1, 参数2):
    """函数说明（可选）"""
    结果 = 参数1 + 参数2    # 函数体（缩进！）
    return 结果, 其他        # 返回（可多个）
```

### 财务类比

> 函数 = Excel 里的自定义函数。输入数据 → 自动算 → 输出结果。
> 
> `def analyze(数据)` = 定义函数  
> `return` = 函数算完往单元格填的结果

### 实战：`calculate_profit` 利润计算函数

```python
def calculate_profit(收入, 成本):
    利润 = 收入 - 成本
    if 利润 > 0:
        状态 = "盈利"
    else:
        状态 = "亏损"
    return 利润, 状态

r1, s1 = calculate_profit(1000000, 800000)  # 利润200,000 盈利
r2, s2 = calculate_profit(500000, 600000)   # 利润-100,000 亏损
```

## 二、lesson08：openpyxl 读写 Excel

### 读：3 行核心代码

```python
from openpyxl import load_workbook
wb = load_workbook("文件路径")    # 打开
ws = wb["Sheet名"]               # 选工作表
value = ws["A1"].value           # 读单元格
```

### 写：3 行核心代码

```python
ws["A1"] = "新内容"              # 写单元格
ws["B7"] = "=SUM(A1:A10)"       # 写公式
wb.save("新文件.xlsx")           # 保存（另存为，不覆盖！）
```

### 用的 API

| 功能 | 写法 |
|------|------|
| 所有Sheet名 | `wb.sheetnames` |
| 选Sheet（按名） | `wb["sheet1"]` |
| 选Sheet（按序号） | `wb.worksheets[0]` |
| 按行列读取 | `ws.cell(row=6, column=2).value` |
| 写公式 | `ws.cell(row=7, column=4).value = "=(B7-C7)/ABS(C7)"` |

## 三、AI 协作实战：4 家公司批量加变动率列

### 工作流

```
幸幸提需求 → DeepSeek 写初稿 → 老财审计 → 修正代码 → 幸幸验收
```

### DeepSeek 代码的 3 个坑

| 坑 | 实际情况 | 修正 |
|----|---------|------|
| 往 D 列写变化率 | D 列是负债数据（双栏布局） | 用 G/H 空列 |
| 表头放 Row 6 | Row 4 才是表头 | 改到 Row 4 |
| 忽略双栏布局 | A-C 是资产，D-F 是负债 | 同时算资产和负债 |

### 修正后结果

```
✅ 101/207/302/503 四个文件全部处理成功
✅ G 列 = 资产变动率 = (B-C)/ABS(C)
✅ H 列 = 负债变动率 = (E-F)/ABS(F)
✅ 原文件数据完好无损
```

### ABS() 函数说明

`ABS()` = 绝对值，防止分母为负数时正负错乱。

| 不用 ABS | 用了 ABS |
|----------|---------|
| (-50 - (-100))/(-100) = -50% | (-50 - (-100))/100 = +50% ✅ |
