## 第4天（2026-06-19）超级学习日 ⚡

### 今日完成内容（休息日全天冲刺）

#### Power BI DAX 进阶
- **SUMX**：逐行计算毛利润 `SUMX('月度收入', [收入]-[成本])`
- **DIVIDE**：安全除法计算毛利率 `DIVIDE([毛利润], SUM([收入]), 0)`
- **RANKX**：按毛利润排名（含排坑经验）
  - ❌ 错误写法：`RANKX(ALL('月度收入'), ...)` → 所有公司都排第1
  - ✅ 正确写法：`RANKX(ALL('公司信息'[公司名称]), [毛利润], , DESC)`
  - 口诀：RANKX第一个参数 = "在哪个名单里排名"
