## 场景背景

海澜餐饮（母公司）给绿野茶饮（子公司）垫付货款50万：
- 海澜账上：应收账款-绿野茶饮 **500,000**
- 绿野账上：应付账款-海澜餐饮 **500,000**

合并报表时，这是一笔内部往来，**不能直接相加**（否则总资产和总负债各多算50万）。

---

## 抵消逻辑

| 科目 | 原金额 | 抵消后金额 | 说明 |
|------|:--:|:--:|:--|
| 应收账款-绿野茶饮 | +500,000 | **-500,000** | 资产减少 |
| 应付账款-海澜餐饮 | +500,000 | **-500,000** | 负债减少 |
| 验证合计 | 1,000,000 | **-1,000,000** | 内部往来对冲清零 |

**抵消分录**：
```
借：应付账款-海澜餐饮  500,000
贷：应收账款-绿野茶饮  500,000
```

---

## PQ实操 · 7步流程

### 步骤1-2：导入数据
- 从Excel导入海澜餐饮、绿野茶饮两张表
- 添加自定义列「公司」标记来源

### 步骤3：追加合并
- `主页` → `追加查询` → 合并两张表为24行大表

### 步骤4-5：创建抵消列（核心）
```
抵消后金额 = 
if [科目] = "应收账款-绿野茶饮" then -[金额]
else if [科目] = "应付账款-海澜餐饮" then -[金额]
else [金额]
```

### 步骤6：添加抵消说明
```
抵消说明 = 
if [科目] = "应收账款-绿野茶饮" then "内部往来抵消-应收"
else if [科目] = "应付账款-海澜餐饮" then "内部往来抵消-应付"
else "正常科目"
```

### 步骤7：Excel验证
```excel
=SUMIFS(D:D,E:E,"内部往来抵消*")  // 结果 = -1000000 ✓
```

## 关键技术点

| 技术 | 作用 |
|------|------|
| 自定义列（公司标记） | 区分数据来源，便于追溯 |
| if...else 条件列 | 识别内部科目并转换符号 |
| 抵消说明列 | 审计追踪，明确抵消原因 |
| SUMIFS验证 | 确认内部往来对冲清零 |
