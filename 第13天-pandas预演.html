⚡ Power Query 操作 vs Pandas 代码 对照

🔘 Power Query（你刚才的操作）

  • 18个步骤，点了几十下鼠标
  • 从文件夹导入 → 展开 → 筛选
  • 拆双栏左/右 → 清洗 → 添加列
  • 复制3个查询 → 追加合并
  • 分组汇总 → 平账验证
  • 耗时：约30分钟

🐍 Pandas（同一件事的代码版）

  • 80行代码，一秒跑完
  • for循环读取5个文件
  • df.iloc[左栏]/df.iloc[右栏] → concat
  • pd.concat → groupby('科目').sum()
  • 资产==负债+权益? → 输出html
  • 耗时：<1秒
# 核心代码片段：这就是你点了18步做的事

for fname in sorted(os.listdir(SRC)):
    df = pd.read_excel(fname, header=None)
    if is_flat:
        # 扁平格式：直接取前3列
        df = df.iloc[:, :3]
    else:
        # 双栏格式：左右拆分
        df_left = df.iloc[:, [0,1,2,3]] # 资产栏
        df_right = df.iloc[:, [5,6,7,8]] # 负债权益栏
        df = pd.concat([df_left, df_right])
